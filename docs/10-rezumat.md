## Harta completă a metabolismului

### Modul STOCARE: insulină ridicată, după mese

```text
CARBOHIDRAȚI:
Glucoză -> prioritate: energie acum (creier, organe)
           -> glicogeneză -> glicogen în ficat + mușchi
           -> lipogezeă (DNL) -> GRĂSIME (doar surplusul final)

GRĂSIMI:
Acizi grași -> esterificare -> trigliceride -> ADIPOCITE
   (LPL activă, HSL blocată de insulină)

PROTEINE:
   Aminoacizi -> sinteză proteică -> MUȘCHI / organe / enzime
              -> surplus: deaminare -> uree + energie/grăsime (rar)
```

### Modul ARDERE: insulină scăzută, post/efort/deficit

```text
GLICOGEN:
   Glicogenoliză -> glucoză -> glicoliză -> ATP + CO2 + apă

GRĂSIME:
   Lipoliză (HSL activă) -> acizi grași -> beta-oxidare -> ATP + CO2 + apă

MUȘCHI, de evitat:
   Proteoliză musculară -> aminoacizi -> gluconeogeneză -> ATP
    Apare cu: efort intens + glicogen gol + proteine insuficiente
```

### Modul URGENȚĂ: glicogen gol + post prelungit

```text
Gluconeogeneză: proteine din dietă sau mușchi + glicerol -> glucoză nouă
Cetogeneză: acizi grași -> corpi cetonici, combustibil alternativ pentru creier
```

## Conversii posibile

```text
Glucoză  -> Grăsime         (lipogezeă)
Proteine -> Glucoză         (gluconeogeneză)
Proteine -> Grăsime         (doar surplus mare, rar)
Grăsime  -> Cetone          (cetogeneză)
 Grăsime  -> Glucoză        (DOAR glicerolul, cantitate mică)
Grăsime  -> Glicogen        (nu se poate întâmpla; limită biochimică)
Glucoză  -> Proteine        (lipsesc aminoacizii esențiali)
```

## Factori care blochează arderea grăsimilor

**În ordine de impact:**

1. **Alcool**: cea mai mare prioritate de ardere, oprește arderea grăsimilor
2. **Somn insuficient**: întregul mecanism se degradează: cortizol, grelină, leptină, insulină
3. **Stres cronic**: cortizol -> glicemie -> insulină, chiar fără mâncare
4. **Insulină cronic ridicată**: zahăr toată ziua, nicio fereastră de ardere
5. **Sedentarism**: mușchi rezistenți la insulină, glicogen mereu plin
6. **Gustări frecvente**: insulina nu scade niciodată
7. **Rezistență la insulină**: blocare permanentă a grăsimii

## Cele 5 idei centrale

### 1. Insulina este comutatorul

```text
Ridicată = stocare (LPL activă, HSL blocată)
Scăzută  = ardere (HSL activă, lipoliză pornită)
```

Controlează insulina prin ce și când mănânci, și influențezi compoziția corporală.

---

### 2. Deficitul caloric este regula #1

```text
Insulină scăzută = ușa este deschisă pentru ardere
Deficit caloric = motivul pentru care corpul trece prin ea
```

Ai nevoie de amândouă. Insulina scăzută fără deficit nu cauzează pierdere de grăsime. Un deficit fără ferestre de insulină scăzută tot funcționează, dar se simte mai greu.

---

### 3. Proteinele sunt prioritatea în deficit

- Nu se stochează: pool mic, distribuie pe parcursul zilei
- Aproape nu devin niciodată grăsime
- Protejează și construiesc mușchi
- Sunt cel mai sățios macronutrient
- Au cel mai mare efect termic

```text
Țintă: 2-2,2g/kg greutate corporală/zi, distribuit în 3-4 mese
```

---

### 4. Nu alegi de unde dispare grăsimea

- Genetica și hormonii decid distribuția
- Grăsimea abdominală pleacă ultima subcutanat, dar grăsimea viscerală periculoasă pleacă prima
- Exercițiile locale nu ard grăsimea din zona antrenată: reducerea punctuală este un mit
- Soluție: deficit + timp + somn + stres redus

---

### 5. Mișcarea și cantitatea sunt cele două pârghii

```text
Mișcarea -> golește glicogenul -> carbohidrații merg la mușchi, nu la grăsime
                               -> crește tamponul caloric zilnic
                               -> îmbunătățește sensibilitatea la insulină

Cantitatea, deficitul -> garantează că arderea netă depășește stocarea
```

Fără mișcare, glicogenul este mereu plin și surplusul devine grăsime. Fără deficit, chiar și insulina scăzută nu face grăsimea să dispară.

## Formula recompozitiei

```text
DEFICIT MIC (-200-300 kcal)
   + PROTEINE RIDICATE (2-2,2g/kg), distribuite
   + Antrenament cu GREUTĂȚI cu progresie, 3-4x/săptămână
   + SOMN bun, 7-9h
   ----------------------------------------------------------------------------------
   = Pierdere de grăsime + câștig muscular în același timp
```

> Simplu și constant bate complex și perfect. 3-4 luni de aplicare corectă pot crea o transformare vizibilă și reală.

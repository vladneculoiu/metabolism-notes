# Stocare

_Ficat, mușchi și adipocite: cum funcționează fiecare și ce decid despre grăsimea corporală_

## Cele două rezerve de glicogen

Glicogenul este stocat în **două locuri complet separate**, cu roluri diferite:

### Glicogenul hepatic (ficat)

```
Rol: REZERVOR PENTRU ÎNTREG CORPUL

Ficat → glicogenoliză → glucoză → SÂNGE → întreg corpul
   (creier, organe, mușchi inactivi)

Se golește:
   → peste noapte, fără mâncare
   → între mese
   → în orice perioadă fără carbohidrați
```

### Glicogenul muscular (mușchi)

```
Rol: COMBUSTIBIL EXCLUSIV PENTRU MUȘCHIUL RESPECTIV

Mușchi → glicogenoliză → glucoză → arsă LA FAȚA LOCULUI
   (NU intră în sânge, NU ajută alte organe sau mușchi)

Se golește:
   → DOAR în timpul efortului fizic al mușchiului respectiv
   → rămâne intact dacă nu te antrenezi
```

{% hint style="warning" icon="head-side-brain" %} Mușchilor le lipsește enzima glucozo-6-fosfatază,
care ar permite trimiterea glucozei în sânge. De aceea mușchii nu reglează glicemia — ficatul este
singurul care poate face asta. {% endhint %}

## Comparație directă

|                          | Glicogen hepatic              | Glicogen muscular           |
| ------------------------ | ----------------------------- | --------------------------- |
| Capacitate               | Mică                          | Mare                        |
| Pentru cine              | **Întreg corpul**             | Doar mușchiul propriu       |
| Trimite glucoză în sânge | **Da** ✅                     | **Nu** ❌                   |
| Se golește când          | Peste noapte, între mese      | În timpul efortului fizic   |
| Se reumple când          | La orice masă cu carbohidrați | Doar dacă efortul l-a golit |

## Cum se umplu ambele rezerve

Se umplu **simultan și independent** — nu în secvență:

```
Glucoză în sânge după masă
        ↓
   ┌────┴────┐
   ↓         ↓
FICAT     MUȘCHI
   rezervele se umplu în paralel, după cât de goale sunt
   │         │
   └────┬────┘
        ↓
   Ambele pline?
        ↓
   DA → surplus → GRĂSIME (lipogeneză)
```

## Starea rezervelor dimineața

Dimineața nu ești complet gol:

| Rezervă           | Stare la trezire                                         |
| ----------------- | -------------------------------------------------------- |
| Glicogen hepatic  | Parțial gol — ficatul a alimentat glicemia toată noaptea |
| Glicogen muscular | Aproape plin — mușchii nu lucrează în somn               |

{% hint style="warning" icon="head-side-brain" %} Dimineața este o fereastră bună de ardere:
insulină scăzută, ficat parțial gol. Tot ai energie pentru activitate și antrenament din glicogenul
muscular intact. {% endhint %}

## Adipocitele: rezerva de grăsime

Adipocitele sunt celule specializate în stocarea trigliceridelor — asemănătoare unor baloane în care
grăsimea ocupă aproape tot spațiul interior.

```
Câștigi grăsime → adipocitele SE UMFLĂ
Pierzi grăsime  → adipocitele SE GOLESC și SE MICȘOREAZĂ
                  (numărul de celule NU scade)
```

{% hint style="warning" icon="head-side-brain" %} De aceea recâștigarea grăsimii este ușoară:
celulele goale rămân acolo, gata să se reumple. Pierderea de grăsime le golește, nu le elimină.
{% endhint %}

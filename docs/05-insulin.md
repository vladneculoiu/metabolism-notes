# 05 -- Insulin

> The central switch that decides, for all macronutrients, whether you store or burn.

## Insulin's role

Insulin is a hormone produced by **pancreatic beta cells** in response to rising blood glucose. It is not bad by itself; it is necessary. Problems appear when it is **chronically elevated**.

```text
HIGH INSULIN (after meals, especially refined carbs)
   -> STORAGE mode
   -> LPL activated (fat enters adipocytes)
   -> HSL blocked (fat cannot leave)
   -> you burn only fresh glucose

LOW INSULIN (fasting, between meals, effort, no sugar)
   -> BURNING mode
   -> HSL activated (fat leaves adipocytes)
   -> beta-oxidation -> ATP
   -> you burn from storage
```

---

## The full daily cycle

```text
PHASE 1 -- YOU EAT
   Carbohydrates -> glucose in blood -> blood glucose rises
   Pancreas -> releases insulin

PHASE 2 -- STORAGE (2-4h after meal)
   Insulin activates LPL -> fat into adipocytes
   Insulin blocks HSL -> fat stays locked
   Glycogenesis -> glucose -> glycogen

PHASE 3 -- TRANSITION (2-4h after meal)
   Blood glucose falls -> insulin gradually falls
   The adipocyte lock loosens

PHASE 4 -- BURNING (low insulin)
   Glucagon/adrenaline -> HSL active
   Lipolysis -> fat from storage
   Beta-oxidation -> ATP
   Final products: CO2 + water
```

> **Daily rule:** how much of the day you spend in Phase 4 (burning) versus Phase 2 (storage) determines whether fat loss happens. A calorie deficit guarantees the balance tilts toward burning, even if you alternate hour by hour.

## How different foods raise insulin

| Raises a lot | Raises moderately | Raises little/none |
| --- | --- | --- |
| Sugar, juice, sweets | Brown rice, whole-grain bread | Meat, fish, eggs |
| White bread, white pasta | Legumes: beans, lentils | Pure fats: oil, butter, avocado |
| White rice, potatoes | Normal fruit: apple, orange | Non-starchy green vegetables |
| Processed cereal, cookies | Dairy: milk, yogurt | Nuts, seeds |
| Very sweet fruit | Protein, moderately | Water, black coffee, tea |

### Combinations change the response

Fat + fiber + protein **slow** carbohydrate absorption, so insulin rises more slowly and less sharply:

```text
White rice alone              -> insulin rises sharply
Rice + chicken breast + veg   -> insulin rises moderately and remains stable
```

## Pancreas vs liver: common confusion

| | Pancreas | Liver |
| --- | --- | --- |
| **Role** | Sensor + commander | Executor |
| **What it does** | Measures blood glucose, releases hormones | Stores or releases glucose |
| **High hormone** | -> insulin (stores) | -> glycogenesis |
| **Low hormone** | -> glucagon (releases) | -> glycogenolysis |
| **Can send glucose into blood** | Not directly | **Yes**, uniquely |

> The liver does **not measure** blood glucose. The pancreas measures and sends commands; the liver executes.

## Insulin and calorie deficit

Common question: *"If insulin is high, does a deficit stop working?"*

**No. The deficit works, but insulin affects moment-to-moment efficiency:**

```text
High insulin for A FEW HOURS/DAY + calorie deficit
   -> at that moment you are not burning fat
   -> the rest of the day compensates
   -> daily NET: you lose fat

ALWAYS high insulin (sugar all day)
   -> the body never gets a proper burning window
   -> fat loss is harder, but still possible with a deficit
```

| Situation | Lose fat? |
| --- | --- |
| Deficit + elevated insulin a few hours/day | Yes |
| Deficit + always elevated insulin | Yes, but harder |
| No deficit + low insulin | No |

> **A calorie deficit is rule #1. Insulin influences how easy and fast it feels.**

## What else blocks fat burning

| Factor | Mechanism |
| --- | --- |
| **Alcohol** | Highest burning priority because it is toxic -> fat burning stops |
| **Poor sleep** | Higher cortisol, higher ghrelin (hunger), lower leptin (satiety), higher insulin resistance |
| **Sedentary lifestyle** | Inactive muscles become insulin resistant -> chronically high insulin |
| **Frequent snacks** | Insulin never drops -> no burning window |
| **Chronic stress** | Cortisol -> higher blood glucose -> higher insulin, even without food |
| **Hypothyroidism** | Lower basal metabolic rate -> fewer calories burned at rest |
| **Artificial sweeteners** | Some may trigger anticipatory insulin responses; controversial and not universal |

> **Hierarchy:** alcohol -> sleep -> stress -> chronic insulin -> sedentary lifestyle. Fix them in that order.

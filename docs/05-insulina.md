# Insulina

_Comutatorul central care decide, pentru toți macronutrienții, dacă stochezi sau arzi_

## Rolul insulinei

Insulina este un hormon produs de **celulele beta pancreatice** ca răspuns la creșterea glicemiei.
Nu este rea în sine — este necesară. Problemele apar când este **cronic ridicată**.

```
INSULINĂ RIDICATĂ (după mese, mai ales carbohidrați rafinați)
   → modul STOCARE
   → LPL activată → grăsimea intră în adipocite
   → HSL blocată  → grăsimea nu poate ieși
   → arzi doar glucoza proaspătă

INSULINĂ SCĂZUTĂ (post, între mese, efort, fără zahăr)
   → modul ARDERE
   → HSL activată → grăsimea iese din adipocite
   → beta-oxidare → ATP
   → arzi din rezerve
```

## Ciclul complet zilnic

```
FAZA 1 — MĂNÂNCI
   Carbohidrați → glucoză în sânge → glicemia crește
   Pancreas → eliberează insulina

FAZA 2 — STOCARE (după masă)
   Insulina activează LPL → grăsimea în adipocite
   Insulina blochează HSL → grăsimea rămâne blocată
   Glicogenogeneza → glucoză → glicogen

FAZA 3 — TRANZIȚIE
   Glicemia scade → insulina scade treptat
   Blocada adipocitelor se slăbește

FAZA 4 — ARDERE (insulină scăzută)
   Glucagon/adrenalina → HSL activă
   Lipoliză → grăsime din rezerve
   Beta-oxidare → ATP
   Produse finale: CO₂ + apă
```

{% hint style="warning" icon="head-side-brain" %} Cât din zi petreci în Faza 4 (ardere) față de Faza
2 (stocare) determină dacă pierzi grăsime. Deficitul caloric garantează că balanța înclină spre
ardere, chiar dacă alternezi oră cu oră. {% endhint %}

## Cum diferite alimente cresc insulina

| Cresc mult                  | Cresc moderat                  | Cresc puțin/deloc                |
| --------------------------- | ------------------------------ | -------------------------------- |
| Zahăr, suc, dulciuri        | Orez brun, pâine integrală     | Carne, pește, ouă                |
| Pâine albă, paste albe      | Leguminoase: fasole, linte     | Grăsimi pure: ulei, unt, avocado |
| Orez alb, cartofi           | Fructe normale: măr, portocală | Legume verzi fără amidon         |
| Cereale procesate, biscuiți | Lactate: lapte, iaurt          | Nuci, semințe                    |
| Fructe foarte dulci         | Proteine (moderat)             | Apă, cafea neagră, ceai          |

{% hint style="warning" icon="head-side-brain" %} **Combinațiile schimbă răspunsul:** grăsimea +
fibrele + proteinele încetinesc absorbția carbohidraților — insulina crește mai lent și mai puțin
brusc. {% endhint %}

```
Orez alb singur               → insulina crește brusc
Orez + pui + legume           → insulina crește moderat și rămâne stabilă
```

## Pancreas vs. ficat

|                                    | Pancreas                             | Ficat                            |
| ---------------------------------- | ------------------------------------ | -------------------------------- |
| **Rol**                            | Senzor + comandant                   | Executor                         |
| **Ce face**                        | Măsoară glicemia, eliberează hormoni | Stochează sau eliberează glucoză |
| **Insulină ridicată**              | → eliberează insulina                | → glicogenogeneză                |
| **Glucagon ridicat**               | → eliberează glucagon                | → glicogenoliză                  |
| **Poate trimite glucoză în sânge** | Nu direct                            | **Da**, în mod unic              |

{% hint style="warning" icon="head-side-brain" %} Ficatul nu măsoară glicemia. Pancreasul măsoară și
trimite comenzi; ficatul execută. {% endhint %}

## Insulina și deficitul caloric

**Nu — deficitul funcționează, dar insulina afectează eficiența moment cu moment:**

```
Insulină ridicată CÂTEVA ORE/ZI + deficit caloric
   → în acel moment nu arzi grăsime stocată
   → restul zilei compensează
   → NET zilnic: pierzi grăsime ✅

Insulină MEREU ridicată (zahăr toată ziua)
   → nicio fereastră de ardere
   → pierderea de grăsime e mai grea, dar posibilă cu deficit ✅
```

| Situație                                  | Pierzi grăsime?     |
| ----------------------------------------- | ------------------- |
| Deficit + insulină ridicată câteva ore/zi | ✅ Da               |
| Deficit + insulină mereu ridicată         | ✅ Da, dar mai greu |
| Fără deficit + insulină scăzută           | ❌ Nu               |

<!-- prettier-ignore -->
{% hint style="warning" icon="head-side-brain" %} 
**Deficitul caloric este regula #1. Insulina
influențează cât de ușor și rapid.** 
{% endhint %}

## Ce altceva blochează arderea grăsimilor

| Factor                      | Mecanism                                                                            |
| --------------------------- | ----------------------------------------------------------------------------------- |
| **Alcool**                  | Prioritate maximă de ardere (e toxic) → arderea grăsimilor se oprește complet       |
| **Somn insuficient**        | Cortizol mai ridicat, foame crescută, sațietate scăzută, rezistență la insulină     |
| **Sedentarism**             | Mușchii inactivi devin rezistenți la insulină → insulină cronic ridicată            |
| **Gustări frecvente**       | Insulina nu scade niciodată → nicio fereastră de ardere                             |
| **Stres cronic**            | Cortizol → glicemie ridicată → insulină ridicată, chiar fără mâncare                |
| **Îndulcitori artificiali** | Unii pot declanșa răspunsuri anticipatorii de insulină — controversat, nu universal |

<!-- prettier-ignore -->
{% hint style="warning" icon="head-side-brain" %} 
**Ierarhie:** alcool → somn → stres → insulină
cronică → sedentarism. Rezolvă-le în această ordine. {% endhint %}

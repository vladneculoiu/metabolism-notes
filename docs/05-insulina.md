# Insulina

_Comutatorul central care decide, pentru toți macronutrienții, dacă stochezi sau arzi_

## Rolul insulinei

Insulina este un hormon produs de **celulele beta pancreatice** ca răspuns la creșterea glicemiei. Nu este rea în sine; este necesară. Problemele apar când este **cronic ridicată**.

```text
INSULINĂ RIDICATĂ (după mese, mai ales carbohidrați rafinați)
   → modul STOCARE
   → LPL activată (grăsimea intră în adipocite)
   → HSL blocată (grăsimea nu poate ieși)
   → arzi doar glucoza proaspătă

INSULINĂ SCĂZUTĂ (post, între mese, efort, fără zahăr)
   → modul ARDERE
   → HSL activată (grăsimea iese din adipocite)
   → beta-oxidare → ATP
   → arzi din rezerve
```

## Ciclul complet zilnic

```text
FAZA 1 -- MĂNÂNCI
   Carbohidrați → glucoză în sânge → glicemia crește
   Pancreas → eliberează insulina

FAZA 2 -- STOCARE (2-4h după masă)
   Insulina activează LPL → grăsimea în adipocite
   Insulina blochează HSL → grăsimea rămâne blocată
   Glicogeneza → glucoză → glicogen

FAZA 3 -- TRANZIȚIE (2-4h după masă)
   Glicemia scade → insulina scade treptat
   Blocada adipocitelor se slăbește

FAZA 4 -- ARDERE (insulină scăzută)
   Glucagon/adrenalina → HSL activă
   Lipoliză → grăsime din rezerve
   Beta-oxidare → ATP
   Produse finale: CO2 + apă
```

{% hint style="warning" icon="head-side-brain" %}
Cât din zi petreci în Faza 4 (ardere) față de Faza 2 (stocare) determină dacă pierderea de grăsime are loc. Un deficit caloric garantează că balanța înclină spre ardere, chiar dacă alternezi oră cu oră.
{% endhint %}

## Cum diferite alimente cresc insulina

| Cresc mult                  | Cresc moderat                  | Cresc puțin/deloc                |
| --------------------------- | ------------------------------ | -------------------------------- |
| Zahăr, suc, dulciuri        | Orez brun, pâine integrală     | Carne, pește, ouă                |
| Pâine albă, paste albe      | Leguminoase: fasole, linte     | Grăsimi pure: ulei, unt, avocado |
| Orez alb, cartofi           | Fructe normale: măr, portocală | Legume verzi fără amidon         |
| Cereale procesate, biscuiți | Lactate: lapte, iaurt          | Nuci, semințe                    |
| Fructe foarte dulci         | Proteine, moderat              | Apă, cafea neagră, ceai          |

### Combinațiile schimbă răspunsul

Grăsimea + fibrele + proteinele **încetinesc** absorbția carbohidraților, astfel insulina crește mai lent și mai puțin brusc:

```text
Orez alb singur                 → insulina crește brusc
Orez + piept de pui + legume   → insulina crește moderat și rămâne stabilă
```

## Pancreas vs ficat: confuzie frecventă

|                                    | Pancreas                             | Ficat                            |
| ---------------------------------- | ------------------------------------ | -------------------------------- |
| **Rol**                            | Senzor + comandant                   | Executor                         |
| **Ce face**                        | Măsoară glicemia, eliberează hormoni | Stochează sau eliberează glucoză |
| **Hormon ridicat**                 | → insulina (stochează)               | → glicogeneză                    |
| **Hormon scăzut**                  | → glucagon (eliberează)              | → glicogenoliză                  |
| **Poate trimite glucoză în sânge** | Nu direct                            | **Da**, în mod unic              |

> Ficatul **nu măsoară** glicemia. Pancreasul măsoară și trimite comenzi; ficatul execută.

## Insulina și deficitul caloric

Întrebare frecventă: _„Dacă insulina este ridicată, deficitul încetează să funcționeze?"_

**Nu. Deficitul funcționează, dar insulina afectează eficiența moment cu moment:**

```text
Insulină ridicată CÂTEVA ORE/ZI + deficit caloric
   → în acel moment nu arzi grăsime
   → restul zilei compensează
   → NET zilnic: pierzi grăsime

ÎNTOTDEAUNA insulină ridicată (zahăr toată ziua)
   → corpul nu primește niciodată o fereastră de ardere propriu-zisă
   → pierderea de grăsime este mai grea, dar tot posibilă cu deficit
```

| Situație                                  | Pierzi grăsime?  |
| ----------------------------------------- | ---------------- |
| Deficit + insulină ridicată câteva ore/zi | Da               |
| Deficit + insulină întotdeauna ridicată   | Da, dar mai greu |
| Fără deficit + insulină scăzută           | Nu               |

{% hint style="warning" icon="head-side-brain" %}
**Deficitul caloric este regula #1. Insulina influențează cât de ușor și rapid se simte.**
{% endhint %}

## Ce altceva blochează arderea grăsimilor

| Factor                      | Mecanism                                                                                                             |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **Alcool**                  | Cea mai mare prioritate de ardere, pentru că este toxic → arderea grăsimilor se oprește                              |
| **Somn insuficient**        | Cortizol mai ridicat, grelină mai ridicată (foame), leptină mai scăzută (sațietate), rezistență mai mare la insulină |
| **Sedentarism**             | Mușchii inactivi devin rezistenți la insulină → insulină cronic ridicată                                             |
| **Gustări frecvente**       | Insulina nu scade niciodată → nicio fereastră de ardere                                                              |
| **Stres cronic**            | Cortizol → glicemie mai ridicată → insulină mai ridicată, chiar fără mâncare                                         |
| **Îndulcitori artificiali** | Unii pot declanșa răspunsuri anticipatorii de insulină; controversat și nu universal                                 |

{% hint style="warning" icon="head-side-brain" %}
**Ierarhie:** alcool → somn → stres → insulină cronică → sedentarism. Rezolvă-le în această ordine.
{% endhint %}

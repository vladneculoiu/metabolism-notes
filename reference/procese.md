# Procese metabolice - Referință

_Toate procesele metabolice relevante într-un singur loc_

## Logica denumirii

- **`-geneză`** = _producere_: construire sau stocare
- **`-liză`** = _descompunere_: eliberare sau ardere

## Tabel complet

| Proces                   | Ce face                          | Substrat -> Produs                     | Se activează când                   | Insulina |
| ------------------------ | -------------------------------- | -------------------------------------- | ----------------------------------- | -------- |
| **Glicoliza**            | Arde glucoza pentru energie      | Glucoză -> ATP + CO2 + H2O             | Mereu, mai mult în efort            | --       |
| **Glicogeneza**          | Stochează glucoza ca glicogen    | Glucoză -> Glicogen                    | După mese, surplus de glucoză       | Ridicată |
| **Glicogenoliza**        | Eliberează glucoza din glicogen  | Glicogen -> Glucoză                    | Între mese, efort, post             | Scăzută  |
| **Lipogeneza (DNL)**     | Transformă surplusul în grăsime  | Glucoză -> Trigliceride                | Glicogen plin, surplus caloric      | Ridicată |
| **Esterificarea**        | Stochează grăsimea în adipocite  | Acizi grași -> Trigliceride            | Insulină ridicată, grăsime în sânge | Ridicată |
| **Lipoliza**             | Eliberează grăsimea din stocare  | Trigliceride -> Acizi grași + glicerol | Insulină scăzută, deficit, efort    | Scăzută  |
| **Beta-oxidarea**        | Arde acizii grași pentru energie | Acizi grași -> ATP + CO2 + H2O         | Insulină scăzută, nevoie de energie | Scăzută  |
| **Gluconeogeneza**       | Produce glucoză din alte surse   | Aminoacizi/glicerol -> Glucoză         | Post, glicogen gol, efort intens    | Scăzută  |
| **Sinteza proteică**     | Construiește mușchi și organe    | Aminoacizi -> Proteine structurale     | Aport proteic + stimul de efort     | Moderată |
| **Proteoliza digestivă** | Digeră proteinele din dietă      | Proteine -> Aminoacizi                 | Orice masă cu proteine              | --       |
| **Proteoliza musculară** | Descompune mușchii               | Mușchi -> Aminoacizi                   | Pool gol + proteine insuficiente    | --       |
| **Cetogeneza**           | Produce combustibil alternativ   | Acizi grași -> Corpi cetonici          | Post prelungit, keto                | Scăzută  |
| **Deaminarea**           | Procesează excesul de aminoacizi | Aminoacizi -> Schelet de carbon + Uree | Surplus de proteine                 | --       |

## Grupate pe mod

### Modul STOCARE: insulină ridicată

```text
Glicogeneza      -> glucoză -> GLICOGEN
Lipogeneza       -> glucoză -> GRĂSIME
Esterificarea    -> acizi grași -> TRIGLICERIDE în adipocite
Sinteza proteică -> aminoacizi -> MUȘCHI
```

### Modul ARDERE: insulină scăzută

```text
Glicogenoliza   -> GLICOGEN -> glucoză -> energie
Lipoliza        -> GRĂSIME -> acizi grași -> energie
Beta-oxidarea   -> acizi grași -> ATP + CO2 + H2O
```

### Modul URGENȚĂ: glicogen gol

```text
Gluconeogeneza    -> aminoacizi/glicerol -> glucoză nouă
Cetogeneza        -> grăsime -> corpi cetonici (combustibil pentru creier)
Proteoliza musculară -> MUȘCHI -> aminoacizi -> glucoză  de evitat
```

## Enzimele cheie

| Enzimă                             | Rol                                  | Controlată de                       |
| ---------------------------------- | ------------------------------------ | ----------------------------------- |
| **LPL** (lipoproteina lipază)      | Pune grăsimea **ÎN** adipocite       | Insulina o **activează**            |
| **HSL** (lipaza hormono-sensibilă) | Scoate grăsimea **DIN** adipocite    | Insulina o **blochează**            |
| **GLUT4**                          | Transportă glucoza în mușchi         | Insulina + contracția musculară     |
| **Glucozo-6-fosfataza**            | Permite eliberarea glucozei în sânge | Prezentă în **ficat**, nu în mușchi |

# Carbohidrați vs Grăsimi vs Proteine

_Toate diferențele esențiale într-un singur loc_

## Comparația principală

| Caracteristică            | Carbohidrați      | Grăsimi                        | Proteine                 |
| ------------------------- | ----------------- | ------------------------------ | ------------------------ |
| **Se descompun în**       | Glucoză           | Acizi grași + glicerol         | Aminoacizi               |
| **Cresc insulina**        | Mult              | Aproape deloc                  | Moderat                  |
| **Cresc glucagonul**      | Nu                | Nu                             | Da, efect de echilibrare |
| **Rezervă tampon**        | Glicogen (~500g)  | Nu, stocare directă ca grăsime | Nu, ~100g pool           |
| **Energie**               | 4 kcal/g          | 9 kcal/g                       | 4 kcal/g                 |
| **Efect termic**          | ~5-10%            | ~2-3%                          | ~25-30%                  |
| **Sațietate**             | Medie             | Bună                           | Cea mai mare             |
| **Viteza de combustibil** | Rapid, exploziv   | Lent, de fundal                | Lent, rezervă de urgență |
| **Rol principal**         | Combustibil rapid | Combustibil lent + hormoni     | Construcție + reparație  |

## Stocare

|                           | Carbohidrați     | Grăsimi                         | Proteine                 |
| ------------------------- | ---------------- | ------------------------------- | ------------------------ |
| **Forma de stocare**      | Glicogen         | Trigliceride în adipocite       | Nu se stochează          |
| **Capacitate de stocare** | ~500g, finit     | Practic nelimitat               | Niciunul                 |
| **Unde**                  | Ficat + mușchi   | Adipocite: subcutanat, visceral | Pool mic: sânge + celule |
| **Stocat direct**         | Prin glicogeneză | Prin esterificare               | Nu                       |
| **Devine grăsime**        | Da, surplusul    | Dacă insulina este ridicată     | Teoretic da, practic nu  |

## Ardere

|                       | Carbohidrați             | Grăsimi                   | Proteine              |
| --------------------- | ------------------------ | ------------------------- | --------------------- |
| **Cum ard**           | Glicoliză → ATP          | Beta-oxidare → ATP        | Gluconeogeneză → ATP  |
| **Când ard**          | Efort intens, mereu ceva | Repaus, efort ușor        | Urgență, glicogen gol |
| **Produse finale**    | CO₂ + apă                | CO₂ + apă                 | CO₂ + apă + uree      |
| **Necesită insulina** | Nu neapărat              | Necesită insulină SCĂZUTĂ | --                    |

## Efectul insulinei în detaliu

| Aliment                    | Răspuns insulinic              |
| -------------------------- | ------------------------------ |
| Zahăr, suc, pâine albă     | Ridicat, rapid                 |
| Orez brun, paste integrale | Moderat-ridicat                |
| Fructe întregi             | Moderat, fibrele îl încetinesc |
| Leguminoase                | Moderat-scăzut                 |
| Carne, pește, ouă          | Scăzut, plus glucagon          |
| Grăsimi pure               | ~ Minimal                      |
| Legume verzi               | ~ Minimal                      |

## Tipuri de efort și combustibilul preferat

| Tip de efort                   | Combustibil principal | Glicogen necesar?              |
| ------------------------------ | --------------------- | ------------------------------ |
| Repaus, somn                   | Grăsime, ~60-70%      | Nu                             |
| Mers, activitate ușoară        | Grăsime + glucoză     | Nu, glicogenul muscular intact |
| Cardio moderat, <60% FCmax     | Mix grăsime + glucoză | Puțin                          |
| Cardio intens, fotbal, sală    | Glicogen, dominant    | **Da**                         |
| Sprint/efort exploziv, <10 sec | Fosfocreatină → ATP   | **Da**, glicogenul urmează     |

# Stocare

> Ficat, mușchi și adipocite: cum funcționează fiecare și ce decid despre grăsimea corporală.

---

## Cele două rezerve de glicogen

Glicogenul este stocat în **două locuri complet separate**, cu roluri diferite:

### Glicogenul hepatic (~100g)

```text
Rol: REZERVOR PENTRU ÎNTREG CORPUL

Ficat -> glicogenoliză -> glucoză -> SÂNGE -> întreg corpul
   (creier, organe, mușchi inactivi)

Se golește:
   -> peste noapte (8h fără mâncare = ficatul aproape gol dimineața)
   -> între mese
   -> în orice perioadă fără carbohidrați
```

### Glicogenul muscular (~300-400g)

```text
Rol: COMBUSTIBIL EXCLUSIV PENTRU MUȘCHIUL RESPECTIV

Mușchi -> glicogenoliză -> glucoză -> arsă LA FAȚA LOCULUI
   (NU intră în sânge, NU ajută alte organe/mușchi)

Se golește:
   -> DOAR în timpul efortului fizic al mușchiului respectiv
   -> rămâne intact dacă nu te antrenezi
```

> **Regula „egoistă":** mușchilor le lipsește enzima glucozo-6-fosfatază, care ar permite trimiterea glucozei în sânge. De aceea mușchii nu reglează glicemia. Ficatul este cel care poate face asta.

---

## Comparație directă

| | Glicogen hepatic | Glicogen muscular |
| --- | --- | --- |
| Capacitate | ~100g, mic | ~300-400g, mare |
| Pentru cine | **Întreg corpul** | Doar mușchiul propriu |
| Trimite glucoză în sânge | **Da** | **Nu** |
| Se golește când | Peste noapte, între mese | În timpul efortului fizic |
| Se reumple când | La orice masă cu carbohidrați | Doar dacă efortul l-a golit |

---

## Cum se umplu ambele rezerve

**Important:** se umplu **simultan și independent**, nu în secvență:

```text
Glucoză în sânge după masă
        |
        +--> FICAT
        |
        +--> MUȘCHI
             rezervele se umplu în paralel în funcție de cât sunt de goale

Ambele pline?
        |
        v
DA -> surplus -> GRĂSIME (lipogeneză)
```

> Mitul „mai întâi ficatul, apoi mușchii" este greșit. Fiecare rezervă extrage glucoză independent în funcție de starea sa. Mușchii goliți de efort trag mai agresiv deoarece exercițiul activează transportorii GLUT4. Doar după ce **ambele** sunt pline surplusul devine grăsime.

---

## Starea rezervelor dimineața

Dimineața **nu** ești complet gol:

| Rezervă | Stare la trezire |
| --- | --- |
| Glicogen hepatic | Parțial gol, circa 60-70% folosit în timpul somnului |
| Glicogen muscular | Aproape plin, deoarece mușchii nu lucrează în timpul somnului |

> **Implicație:** dimineața este o fereastră bună de ardere: insulină scăzută, ficat parțial gol. Dar tot ai energie pentru activitate și antrenament din glicogenul muscular intact.

## Adipocitele: rezerva de grăsime

### Ce sunt

```text
Adipocit = celulă asemănătoare unui balon, specializată în stocarea trigliceridelor
   -> grăsimea ocupă ~90% din celulă
   -> nucleul este împins la margine
```

### Dinamica

```text
Câștigi grăsime -> adipocitele SE UMFLĂ
Pierzi grăsime  -> adipocitele SE GOLESC și SE MICȘOREAZĂ
                   (numărul de celule NU scade)
```

> De aceea recâștigarea grăsimii este ușoară: celulele goale rămân acolo, gata să se reumple. Pierderea de grăsime le golește; nu le elimină.

### Numărul de adipocite

```text
La adulți, numărul este relativ FIX (~10-30 miliarde)
   -> creștere mică de grăsime = celulele existente se umflă
   -> creștere mare de grăsime, odată ce celulele ajung la maxim = corpul crează CELULE NOI
   -> celulele noi nu dispar odată cu pierderea în greutate
```

### Adipocitele nu sunt saci pasivi

Sunt un **organ hormonal activ** și influențează metabolismul prin hormoni:

| Hormon/substanță | Efect |
| --- | --- |
| **Leptina** | Semnalizează „sunt sătul" și reglează foamea |
| **Adiponectina** | Îmbunătățește sensibilitatea la insulină |
| **Substanțe inflamatorii** | Secretate de grăsimea viscerală -> rezistență la insulină |

> Grăsimea viscerală abdominală este periculoasă parțial pentru că acele adipocite eliberează substanțe inflamatorii care perturbă metabolismul.

---

## Tipuri de adipocite

| Tip | Culoare | Rol |
| --- | --- | --- |
| **Grăsime albă** | Albă/gălbuie | Stochează energie, cea mai mare parte a grăsimii corporale |
| **Grăsime brună** | Brun | **Arde** energie pentru a produce căldură |
| **Grăsime bej** | Intermediar | Grăsime albă care poate deveni mai „arzătoare" cu frig/exerciții |

---

## Unde se stochează grăsimea: nu alegi tu

Distribuția este decisă de **genetică + hormoni**, nu de exercițiile locale:

| Factor | Efect |
| --- | --- |
| **Genetica** | Tiparele de bază moștenite din familie |
| **Testosteronul** la bărbați | Favorizează stocarea **abdominală**, forma „măr" |
| **Estrogenii** la femei | Favorizează stocarea **șolduri/coapse**, forma „pară" |
| **Cortizolul** din stres | Favorizează grăsimea viscerală **abdominală** periculoasă |

### Menopauza

Estrogenii scad, deci femeile încep adesea să stocheze mai multă grăsime în jurul abdomenului, similar cu bărbații. Aceasta confirmă rolul direct al hormonului.

---

## Ordinea în care grăsimea pleacă în timpul pierderii în greutate

```text
1. Grăsimea VISCERALĂ din jurul organelor  -> pleacă PRIMA  (bine pentru sănătate, mai puțin vizibilă)
2. Față, gât, brațe                        -> vizibilă rapid
3. Piept, spate
4. Coapse, fese
5. Grăsimea abdominală SUBCUTANATĂ         -> pleacă ULTIMA  (burta vizibilă)
```

> **De ce burta este ultima:** adipocitele subcutanate abdominale au mai mulți receptori alfa, care încetinesc lipoliza, și mai puțini receptori beta, care o pornesc. Frâna este trasă mai puternic acolo.

### Mitul reducerii punctuale

```text
Abdomenele -> NU ard grăsimea abdominală
Genuflexiunile -> NU ard grăsimea de pe coapse
Exercițiile ard grăsime din ÎNTREG corpul, în ordinea pe care o dictează genetica
```

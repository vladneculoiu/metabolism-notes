# Proteine

> De ce nu se stochează, cum protejează mușchii și de ce aportul constant este obligatoriu.

---

## Traseul complet

```text
1. MĂNÂNCI (carne, ouă, pește, lactate, shake proteic)
        
2. DIGESTIE (proteoliză) -> AMINOACIZI
        
3. Ajung la FICAT (managerul) -> distribuiți în corp
        
4. Intră în „POOL-ul de aminoacizi" (mic, ~100g, în sânge + celule)
        
5. Aminoacizii merg în mai multe direcții, pe bază de prioritate:
        ---- SINTEZĂ PROTEICĂ -> mușchi/organe  prioritate
        ---- Enzime, hormoni, anticorpi, sistem imunitar
        ---- Energie dacă e nevoie -> gluconeogeneză -> ATP
        ---- Surplus mare -> DEAMINARE -> azot -> uree (urină) + rest -> ars/grăsime
        
6. RISC dacă pool-ul este gol + nu mănânci proteine:
        ---- PROTEOLIZĂ MUSCULARĂ -> corpul descompune mușchii -> aminoacizi
```

---

## Particularitatea mare: proteinele nu se stochează

Cea mai importantă diferență față de ceilalți macronutrienți:

```text
Carbohidrați -> glicogen (~500g rezervă)           stocare
Grăsimi      -> adipocite (~rezervă nelimitată)    stocare
Proteine     -> fără stocare dedicată              fără rezervor
```

### Pool-ul de aminoacizi: mic și temporar

„Pool-ul" nu este un organ. Este suma **aminoacizilor liberi** din sânge și interiorul celulelor, gestionată de ficat:

- **Mărime:** ~100g aminoacizi liberi total, minuscul față de ~10-12kg proteine structurale din mușchi/organe
- **Durată:** se golește în câteva ore după o masă
- **Locații:** plasmă sanguină (mică, ~5-6g) + lichid intracelular (cea mai mare parte)

```text
Analogie:
   Pool = cont curent (mic, pentru cheltuielile zilnice)
   Mușchi/organe = avere structurală (nu vrei să o vinzi pentru facturi)
```

---

## De ce corpul „ia din mușchi"

```text
Fără aport proteic + corpul are nevoie de aminoacizi
        
Pool-ul mic se golește în câteva ore
        
Nu există rezervă de stocare
        
Singura sursă mare = mușchii (proteine structurale)
        
PROTEOLIZĂ MUSCULARĂ = catabolism
```

> **Soluție:** aminoacizi din dietă înainte ca pool-ul să se golească. De aceea **aportul constant pe parcursul zilei** contează mai mult decât un total dintr-o singură masă.

---

## Distribuția contează mai mult decât totalul

Corpul utilizează eficient circa **~25-40g proteine per masă**:

```text
 4 mese x 30g proteine = 120g distribuite
   -> pool-ul rămâne plin -> mușchii protejați

 1 masă x 120g proteine = risipitor
   -> surplusul nu se stochează -> descompus/eliminat
   -> pool-ul este gol restul zilei -> risc de catabolism
```

---

## Pot proteinele să devină grăsime?

**Teoretic da, practic neglijabil.**

Cale: aminoacizi -> deaminare (eliminare azot) -> schelet de carbon -> lipogeneză -> grăsime.

De ce este rar:

1. **Costisitor energetic**: corpul plătește un cost ridicat și evită asta
2. **Priorități superioare:** mușchi, enzime, energie directă, glucoză vin primele
3. **Efect termic ridicat:** arzi ~25-30% din caloriile proteinelor doar digerându-le
4. **Sațietate puternică:** este greu să mănânci un surplus mare de proteine

> **Concluzie practică:** proteinele nu îngrașă. Un **surplus caloric total** îngrașă, iar acel surplus vine aproape întotdeauna din carbohidrați rafinați + grăsimi, nu din proteine.

---

## Proteine și insulina

Proteinele cresc insulina **moderat**: mai puțin decât carbohidrații, dar nu zero.

```text
Carbohidrați -> creștere puternică a insulinei, glucagon neschimbat -> stocare
Proteine     -> creștere moderată a insulinei + creștere glucagon   -> echilibru
```

Glucagonul compensează parțial efectul de stocare, astfel grăsimea nu este blocată la fel de puternic. De aceea o gustare proteică este mult mai puțin dăunătoare decât una zaharoasă, deși tot întrerupe fereastra de ardere.

---

## Când apare catabolismul muscular

| Situație | Risc de proteoliză musculară |
| --- | --- |
| Repaus, proteine suficiente | Minimal (~2-5% din energie) |
| Efort intens + glicogen **plin** | Scăzut, glicogenul acoperă |
| Efort intens + glicogen **gol** | **Ridicat**, corpul atacă mușchii |
| Post prelungit + proteine insuficiente | **Ridicat** |
| Deficit agresiv + proteine puține | **Ridicat** |

> **Paradox:** repausul este „mai sigur" pentru mușchi decât efortul intens pe nemâncate. Activitatea ușoară/moderată pe nemâncate arde grăsime; efortul intens pe nemâncate poate ataca mușchii.

---

## Particularități

- **Nu se stochează**: fără rezervor, deci aportul trebuie să fie constant pe parcursul zilei
- Cresc insulina **moderat** + cresc glucagonul, un echilibru mai bun decât carbohidrații
- Cel mai **sățios** macronutrient
- Cel mai mare **efect termic**: arzi calorii doar digerându-le (~25-30%)
- Energie: **4 kcal/g**
- Țintă pentru recompozitie: **2-2,2g/kg greutate corporală/zi**, distribuit în 3-4 mese

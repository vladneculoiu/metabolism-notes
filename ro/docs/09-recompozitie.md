# Recompozitia corporală

> Aplicarea practică: cum să pierzi grăsime și să câștigi masă musculară în același timp.

---

## Ce este recompozitia

Recompozitia corporală = pierdere de grăsime + câștig de masă musculară **în același timp**.

Necesită condiții care par opuse:

- **Pierderea de grăsime** -> deficit caloric -> insulină scăzută -> arde grăsime
- **Câștigul muscular** -> disponibilitate energetică -> insulină/anabolism -> construiește mușchi

```text
DEFICIT MARE -> pierdere de grăsime mai rapidă
              -> dar mai puține resurse pentru construirea mușchilor
              -> risc mai mare de pierdere musculară

DEFICIT MIC  -> pierdere de grăsime mai lentă
              -> rămân suficiente resurse pentru construirea mușchilor
```

---

## Cine recompune cel mai ușor

| Factor | Avantaj |
| --- | --- |
| **Începător la ridicarea de greutăți** | Cel mai mare avantaj, „câștigurile de începător" |
| **Are grăsime de pierdut**, 3+ kg | Avantaj mare: grăsimea furnizează energie pentru construcție |
| Revenire după o pauză | Avantaj mare: memoria musculară |
| Avansat și deja slab | Avantaj mic: fazele de masă/definire funcționează adesea mai bine |

## Cele 4 piloane esențiale

### 1. Deficit caloric mic

```text
Pierdere pură de grăsime:   deficit mare (-500-700 kcal)
RECOMPOZITIE:               deficit mic  (-200-300 kcal)
Faza de masă musculară:     surplus      (+200-300 kcal)
```

Un deficit mic este suficient pentru a pierde grăsime, lăsând resurse pentru construirea mușchilor.

### 2. Proteine: 2-2,2g/kg greutate corporală/zi

Prioritate absolută. **Distribuite** pe parcursul zilei, nu mâncate într-o singură masă:

```text
80kg greutate corporală -> 160-176g proteine/zi
   Exemplu: 4 mese x 40-44g proteine
```

Proteinele:

- Protejează mușchii de catabolism în deficit
- Construiesc mușchi noi prin sinteză proteică
- Sunt cel mai sățios macronutrient
- Nu se stochează aproape niciodată ca grăsime

### 3. Antrenament cu greutăți: 3-4x/săptămână

**Cel mai important stimul.** Fără el, proteinele nu construiesc mușchi deoarece „comanda" lipsește.

- Exerciții compuse: genuflexiuni, împins la piept, tracțiuni, îndreptări
- **Progresie**: creșterea greutăților/repetărilor -> stimul de creștere continuu
- Cardio moderat, nu excesiv, deoarece prea mult poate crește cortizolul

### 4. Somn: 7-9 ore

Construirea mușchilor și recuperarea au loc **în timpul somnului**, nu în sală.

- ~60-70% din testosteron este produs noaptea
- O săptămână de somn insuficient poate reduce testosteronul cu ~10-15%
- Lipsa somnului crește cortizolul -> stocare abdominală + descompunere musculară

## Combustibil în funcție de tipul de efort

| Situație | Ce arde | Mușchi |
| --- | --- | --- |
| Repaus / efort ușor | Grăsime ~60-70% | Protejat |
| Efort intens + glicogen plin | Glicogen, exploziv | Protejat |
| Efort intens + glicogen gol | Ceva grăsime + **MUȘCHI** | Atacat |

> **Paradox important:** repausul și efortul ușor ard mai multă grăsime și protejează mușchii mai bine decât efortul intens pe nemâncate.

---

## Strategia în jurul antrenamentului

### Înainte de efort intens

```text
45-60 min înainte:
   Carbohidrat rapid (banană/curmale/fructe) -> ~25-50g carbohidrați
   -> reumple glicogenul muscular la timp
   -> mușchii au combustibil -> nu se atacă singuri

Nu este necesar pentru:
   - efort ușor/moderat: mers, pilates, cardio ușor
   - aceste eforturi ard grăsime direct
```

### După efort intens

```text
În primele 1-2h, fereastra anabolică:
Proteine: 30-40g -- PRIORITATE, repară mușchii
   Carbohidrați: 40-60g -- reumple glicogenul epuizat

De ce carbohidrații sunt „iertători" după efort:
   -> glicogenul gol absoarbe carbohidrații direct în mușchi
   -> insulina lucrează în favoarea ta, anti-catabolică
   -> aproape nimic nu merge la grăsime deoarece glicogenul are spațiu
```

### Dacă nu te antrenezi intens a doua zi

Poți sări carbohidrații post-efort dacă obiectivul este pierderea de grăsime și nu te antrenezi mâine:

```text
Fără carbohidrați după efort moderat + proteine suficiente:
   -> insulina rămâne scăzută -> arderea grăsimilor continuă
   -> glicogenul se reumple lent prin gluconeogeneză în ~48h
   -> mușchii sunt protejați de proteinele din dietă
```

---

## Cum se leagă cu insulina

```text
ZILELE DE ANTRENAMENT:
   Carbohidrați în jurul efortului -> insulină ridicată -> glicogen -> mușchi

ZILELE DE REPAUS:
   Carbohidrați moderate/reduse -> insulină scăzută -> arderea grăsimilor
   Proteine constante -> mușchi protejați
```

## Creatina, supliment opțional

Cel mai bine documentat supliment pentru forță și masă musculară. **Nu este obligatoriu**: elementele de bază creează 90% din rezultat.

### Ce face

```text
Creatina -> stocată în mușchi ca FOSFOCREATINĂ
   -> regenerează rapid ATP pentru explozii scurte: sprinturi, ridicări
   -> mai multă energie disponibilă -> antrenament mai intens -> stimul mai puternic
```

### Beneficii dovedite

- Forță și putere crescute
- Masă musculară mai mare deoarece permite antrenament mai intens
- Recuperare mai rapidă
- Păstrarea mai bună a forței în deficit, protejând mușchii

### Cum se ia

```text
3-5g creatină MONOHIDRAT/zi, constant
   -> inclusiv în zilele de repaus
   -> nu este necesară o fază de încărcare
   -> după ~3-4 săptămâni, rezervele musculare sunt pline
```

> **Notă despre cântar:** +1-2kg în primele săptămâni este apă intramusculară, nu grăsime. Mușchiul hidratat este mai funcțional.

### Forma corectă

```text
Creatina MONOHIDRAT -> cea mai studiată, mai ieftină
Forme non-monohidrat sau „fancy" (HCl, tamponată etc.) -> mai scumpe, fără avantaj dovedit
```

## Așteptări realiste

```text
Recompozitia este LENTĂ, mai lentă decât pierderea simplă de grăsime
   -> cântarul poate să se miște puțin sau deloc
      (grăsime pierdută + mușchi câștigați = se anulează pe cântar)
   -> dar oglinda, talia și forța se schimbă vizibil
```

### Indicatori de urmărit, nu doar greutatea pe cântar

| Indicator | Ce arată |
| --- | --- |
| **Circumferința taliei** | Pierderea de grăsime viscerală/abdominală |
| **Oglinda** | Tonusul și forma |
| **Greutățile la antrenament** | Câștigul muscular |
| **Hainele** | Cea mai practică măsură |
| **Fotografii lunare** | Progres vizual obiectiv |

### Timeline realist

```text
1 lună:     antrenamentul se îmbunătățește, mai multă forță
2 luni:     hainele se simt diferit, talia scade
3-4 luni: -3-4kg grăsime + 1-2kg mușchi = transformare vizuală clară
```

# Carbohidrați

> De la furculiță la ATP: traseul complet al glucozei prin corp.

---

## Traseul complet

```text
1. MĂNÂNCI (pâine, orez, fructe, zahăr)
        
2. DIGESTIE -> carbohidrații se descompun în GLUCOZĂ
        
3. Glucoza intră în sânge -> glicemia crește
        
4. Pancreasul o detectează -> eliberează INSULINA
        
5. Insulina direcționează glucoza, în ordine de prioritate:
        ---- ENERGIE ACUM (creier, organe, mușchi)  prioritatea #1
        ---- GLICOGENEZA -> glicogen în ficat + mușchi
        ---- LIPOGENEZA -> grăsime (doar surplusul final)
        
6. ARDERE (glicoliză):
Glucoză -> mitocondrie -> ATP
   Produse finale: CO2 + apă
```

---

## Ordinea de prioritate contează

Glucoza **nu** merge direct la stocare ca grăsime. Ordinea reală este:

| Prioritate | Destinație | Cine o folosește |
| --- | --- | --- |
| **#1** | Energie imediată | Creier (~120g/zi), organe, întreținere celulară -- non-stop |
| **#2** | Reumplere glicogen | Ficat + mușchi, dacă există loc (~500g total) |
| **#3** | Grăsime | Doar surplusul final, după ce #1 și #2 sunt acoperite |

> „Tamponul" zilnic pentru o persoană sedentară este în jur de ~200g de carbohidrați: energie + reumplere ficat fără a deveni grăsime. Activitatea fizică crește semnificativ acest tampon.

## Glicemia și insulina: termostatul

```text
PANCREASUL = senzor (termostat)
   -> detectează glicemia -> trimite comenzi prin hormoni

FICATUL = executor
   -> primește comenzile -> stochează sau eliberează glucoză
```

| Situație | Pancreas | Ficat |
| --- | --- | --- |
| Glicemia crește după masă | Eliberează **insulina** | Stochează glucoza ca glicogen |
| Glicemia scade între mese | Eliberează **glucagonul** | Eliberează glucoza din glicogen |

## Zahărul rafinat: ce este și de ce contează

Zahărul procesat industrial, lipsit de fibre, vitamine și minerale. Fără fibre -> absorbție instantanee -> vârf brusc de insulină.

### Diferența cheie: fibrele

```text
Măr întreg   -> fibrele încetinesc absorbția -> insulina crește lent
Suc de mere  -> fără fibre -> insulina crește brusc (= zahăr lichid)
```

### Test simplu de recunoaștere

> *„Aș putea mânca asta în forma sa naturală?"*
>
> - Trestie de zahăr crudă -> da, conține multe fibre
> - O linguriță de zahăr alb -> instant, pur -> rafinat

### Forme comune de evitat

**Evidente:** zahăr alb/brun, miere procesată, sirop glucozo-fructozic, sirop de porumb bogat în fructoză (HFCS), sirop de agave.

**Ascunse în produse:** sucuri, băuturi energizante, cereale de mic-dejun, batoane „proteice", iaurturi cu fructe, ketchup, sosuri dulci, pâine albă, produse de patiserie.

**Capcana de pe etichetă:** zahărul se ascunde sub zeci de nume:
> sirop de glucoză, dextroză, maltoză, zaharoză, suc de trestie evaporat, maltodextrină, concentrat de suc de fructe...

### Ce nu este zahăr rafinat

- Fructe întregi, cu fibrele lor
- Carbohidrați complecși: ovăz, orez brun, cartof dulce, leguminoase
- Legume

## Glicogenul: sistemul de stocare

| | Glicogen hepatic | Glicogen muscular |
| --- | --- | --- |
| Capacitate | ~100g | ~300-400g |
| Pentru cine | **Întreg corpul** | Doar mușchiul respectiv |
| Poate trimite glucoză în sânge | **Da** | **Nu** |
| Se golește | Peste noapte, între mese | Doar în timpul efortului |
| Se reumple | La orice masă cu carbohidrați | Doar după efortul care l-a golit |

> **Cele două rezerve se umplu simultan și independent**, nu în secvență („mai întâi ficatul, apoi mușchii"). Fiecare extrage glucoză în funcție de cât de goală este. Doar după ce **ambele** sunt pline surplusul devine grăsime.

### Cât durează să se golească

| Situație | Timp |
| --- | --- |
| Repaus complet, fără mâncare | 24-36h |
| Somn (8h) | Golește glicogenul **hepatic**, deoarece ficatul menține glicemia |
| Antrenament intens (sprinturi, fotbal) | 1-2h pot goli mare parte din glicogenul **muscular** |

### Cât durează să se reumple după efort

```text
Carbohidrați după sport  -> reumplere completă în ~24h
Fără carbohidrați        -> reumplere lentă, parțială din proteine (~48-72h)
```

## Gustările și fereastra de ardere

Fiecare gustare **repornește Faza 1**: insulină ridicată, arderea grăsimilor întreruptă.

```text
8:00  mic-dejun -> insulină ridicată
10:30 gustare   -> insulină ridicată DIN NOU (fără timp să ardă)
13:00 prânz     -> insulină ridicată
16:00 gustare   -> insulină ridicată DIN NOU
-> corpul nu intră niciodată în modul ardere
```

**Ideal pentru recompozitie:** 2-3 mese clare, fără gustări -> ferestre de 4-5h în care insulina scade și arderea grăsimilor poate avea loc.

**Dacă ai nevoie de o gustare:** proteine + grăsimi, cum ar fi ouă, nuci sau brânză. Nu cresc insulina agresiv.

## Particularități

- Singurul macro cu o **rezervă tampon** dedicată: glicogenul
- Cresc insulina **cel mai mult** dintre toți macronutrienții
- Combustibil **rapid** pentru efort intens/exploziv: sprinturi, ridicare de greutăți
- **Rafinat** = vârf brusc; **complex** = absorbție mai lentă și mai uniformă
- Energie: **4 kcal/g**
